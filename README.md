## About the Project 📃

Sunflower Kanban Board is a lightweight task management application inspired by the Kanban workflow methodology. The project allows users to organize tasks into different workflow stages using an intuitive drag-and-drop interface.

Built with HTML, CSS, and JavaScript, this project demonstrates drag-and-drop functionality, and clean user interface design without relying on external frameworks.

---

## Project Overview 📊

Summarize the project objectives, key metrics, and relevant information.

- **Objective:** Build a simple Kanban board application for managing tasks through drag-and-drop interactions.
- **Target Audience:** Students, beginner developers, freelancers, and anyone looking for a simple task management solution.
- **Key Metrics:**
  - Drag & Drop Task Management
  - Multiple Task Columns
  - Interactive User Experience
  - Lightweight Application
  - Pure JavaScript Implementation
- **Deployment:** Hosted on GitHub Pages.

## Installation ⚙️

Clone the repository:

```bash
git clone https://github.com/torufa/Sunflower-kanban-board.git
```

Navigate to the project folder:

```bash
cd Sunflower-kanban-board
```

Open the project:

```bash
Open index.html
```

Or run using **VS Code Live Server**.
